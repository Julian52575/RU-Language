Coding-byte

1 byte / 8 bits.

Les instructions n’ayant que des opérandes d’un seul type possible n’ont pas d’opcode. Exemple: UNSET_RETURN

valuemeaning
0b00Non existant
0b01Inutilisé
0b10Constante
0b11ID variable
Example:

0x1B ou 0b 00 01 10 11.

Operand 4 : 11 donc variable id.

Operand 3 : 10 donc constant.

Operand 2 : 01 unused.

Operand 1 : 00 donc inexistant.

Operands
Constante

Valeur constante.

Identifiant de variable

Chaque variable a un identifiant unique, la Nème variable créée à l’identifiant n.

Lorsque l’on sort du scope d’une fonction, toutes les variables présente dans cette scope sont effacées et l’identifiant diminue.

Revoit une exception si la variable n’a pas été définie.
